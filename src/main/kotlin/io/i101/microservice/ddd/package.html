Integration guides

- Named project, e.g youtube-app - pom.xml groupId => io.i101.youtube artifactId => youtube-app version => should start with 1.0.1-SNAPSHOT, e.g. youtube-app-1.0.1-SNAPSHOT

- Named module, e.g. streaming-microservice - Rename io.i101.microservice.ddd to io.i101.[project name].[microservice name] E.g A base package of streaming microservice - youtube should be io.i101.youtube.streaming - pom.xml groupId => io.i101.youtube artifactId => streaming-microservice version => should start with 1.0.1-SNAPSHOT, e.g. streaming-microservice-1.0.1-SNAPSHOT
